# chromium_addon

Home Assistant Add-on that runs **headless Chromium** and exposes a **Chrome DevTools Protocol (CDP)** debug port.

## Features

- Headless Chromium
- Remote debugging via CDP (`9222`)
- Playwright / Puppeteer compatible
- No UI, no API, minimal and clean
